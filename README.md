# 🔗 LinkVault

LinkVault is a modern bookmark manager built using Firebase that allows users to save, organize, and access useful links from anywhere.

## 🚀 Features

- 🔐 User Authentication (Firebase Auth)
- 🔗 Save and manage bookmarks (URL, title, description)
- 📂 Categorize links (e.g., DSA, Web Dev, AI)
- ⭐ Mark favorite links
- 🔍 Search and filter bookmarks
- ⚡ Real-time updates using Firestore
- 🗑️ Edit and delete bookmarks

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Firebase (Firestore + Auth)
- Styling: CSS / Tailwind
